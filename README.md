# Html2mobi

将搜狐读书频道html页面转换成mobi格式

依赖KindleGen

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'html2mobi'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install html2mobi

## Usage

html2mobi url [path]
